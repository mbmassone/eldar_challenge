
# Challenge Eldar

## Requisitos

- Node.js
- npm
- React


## Instalación

1. Instala las dependencias:
   ```bash
   npm install
   ```

2. Para compilar el proyecto:
   ```bash
   npm run build
   ```

3. Para levantar el servidor en modo desarrollo:
   ```bash
   npm run dev
   ```

   El proyecto estará disponible en: [http://localhost:5173](http://localhost:5173)
